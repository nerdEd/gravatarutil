= gravatarutil

www.nerdEd.net

== DESCRIPTION:

A ruby gravatar utility.

== FEATURES/PROBLEMS:

+ Retrieves gravatar image for e-mail
+ Can save gravatar image to user defined file
- Can't set OS X icon
- No persistent configuration

== SYNOPSIS:

Usage: 

# Places gravatar image for FrankNBeans@gmail.com in working directory.
gravatarutil FrankNBeans@gmail.com

# Saves gravatar image for Molly@gmail.com as file specified
gravatarutil Molly@gmail.com -f ~/Users/Molly/gravatarImage.jpg
	

== REQUIREMENTS:

none

== INSTALL:

sudo gem install gravatarutil
